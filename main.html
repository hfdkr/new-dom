Exercice 5 : Accordion / FAQ

HTML est un langage utilisé pour structurer une page web.

CSS sert à styliser la page web, comme les couleurs et la mise en page.

JavaScript permet d'ajouter de l'interactivité sur une page web.

Le DOM représente la structure HTML que JavaScript peut modifier.

localStorage permet de garder des données même après le rechargement de la page.
